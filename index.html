我的第一篇博客：测试 GitHub Pages 发布

作者：Gove | 发布时间：2026年5月14日 | 分类：Web 前端

大家好！这是我利用 GitHub Pages 搭建的个人博客的第一篇文章。本次实验的主要目的是了解 GitHub 的基本使用方式，并掌握 Web 项目中绝对路径和相对路径的概念。

为了完成实验要求，我在这篇文章中插入了两张图片，分别使用了不同的路径引用方式：

图 1：使用绝对路径引用的网络图片 (http://...)

上面这张图片使用的是绝对路径，直接指向了互联网上的一个真实图片地址。无论这个 HTML 文件被放在哪里，只要有网络，图片就能正常显示。

图 2：使用相对路径引用的本地图片 (image/1.png)

上面这张图片使用的是相对路径。image/1.png 表示在当前 HTML 文件所在的同级目录中，进入 image 文件夹找到 1.png。这种方式非常适合引用项目内部的资源文件。

通过这次实验，我不仅学会了如何美化网页，还成功将博客部署到了线上。期待未来能在这里分享更多有趣的技术文章！
